Параграф p

Тег quote это кавычки
Тег adress используется для обозначения автора цитаты или текста, например: Miha

blockquote используется для обозначения большой цитаты, в теге blockquote можно указать атрибут "site" в котором можно указать ссылку на сайт откуда взята цитата

Тег time используется для указания времени 1900-11-12

Теги не имеющие семантического веса

Жирный текст тег b
Курсивный текст тег i
Машинописный текст тег tt
Зачеркнутый текст тег s

Теги имеющие семантический вес

Жирный текст тег strong
Курсивынй текст тег em
Зачеркнутый текст тег del
Подчеркнутый текст тег ins
Тег code не сохраняет форматирование текста, используется для отображения кода на странице 
        Тег pre сохраняет форматирование текста,
        так же используется для отображения кода на странице

Тег small делает текст в предложении меньше

Тег big делает текст в предложении больше

Тег sub отображает текст в нижнем индексе

Тег sup отображает текст в верхнем индексе
