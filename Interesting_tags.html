Тег abbr используется для обертки аббревиатуры, в атрибуте type указывается расшифровка аббревиатуры, например DRY обернут в такой тег

Тег datalist позволяет добавить поле с опциями, создается так: 

        
            input list="list" 
            datalist id="list"

Тег meter используется для изображения прогресса на разных шкалах


Тег progress используется для изображения прогресса на одной шкале

Для маркировки текст можно обернуть в тег mark

Тег canvas используется для рисунков и прочихх подвижных объектов, управляется с помощью JS
